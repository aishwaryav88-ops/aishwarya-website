# Aishwarya Vijayan Academic Website

A clean static academic website for GitHub Pages.

## Files

- `index.html` — website content and structure
- `styles.css` — responsive visual design
- `script.js` — mobile menu and dynamic year
- `assets/Aishwarya_Vijayan_CV.pdf` — downloadable CV
- `.nojekyll` — tells GitHub Pages to serve the site as plain static files

## Publish with GitHub Pages

1. Create a new GitHub repository, for example `aishwarya-website`.
2. Upload all files in this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch `main` and folder `/root`.
6. Save. GitHub will publish the site at a URL like:
   `https://YOUR-GITHUB-USERNAME.github.io/aishwarya-website/`

## Customize before publishing

- Replace the initials avatar with a real headshot if desired.
- Update publication links as new papers are added.
- Replace the CV PDF whenever the CV changes.
- Add a custom domain later by configuring GitHub Pages and adding a `CNAME` file.
